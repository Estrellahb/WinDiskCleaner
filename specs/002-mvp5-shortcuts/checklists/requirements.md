# Requirements Checklist: MVP5 快捷方式检测与修复

## Scope

- [x] 扫描桌面、开始菜单、启动项快捷方式
- [x] 递归扫描 `.lnk` 文件
- [x] 解析快捷方式目标路径和参数
- [x] 检查目标是否存在
- [x] 标记失效快捷方式
- [x] 支持取消扫描
- [x] 支持批量删除选中失效项
- [x] 支持 CSV 报告导出
- [x] 支持 HTML 报告导出

## UI

- [x] Tab5 快捷方式页面
- [x] 开始扫描按钮
- [x] 进度条
- [x] 扫描统计：总计 / 有效 / 失效
- [x] 快捷方式列表：名称 / 路径 / 目标路径 / 状态 / 来源目录
- [x] 失效项红色高亮
- [x] 搜索过滤输入框
- [x] 复选框，支持全选/反选
- [x] 删除选中按钮，删除前弹窗确认
- [x] 导出报告按钮
- [x] 删除后刷新列表

## Safety

- [x] 只处理 `.lnk` 快捷方式文件本身
- [x] 不删除目标程序或文件
- [x] 删除前展示路径和目标
- [x] 默认只删除失效快捷方式
- [x] 有效快捷方式不进入删除候选

## Non-goals

- [x] 不做注册表快捷方式清理
- [x] 不修复或重建快捷方式目标路径
- [x] 不扫描开始菜单以外的自定义位置
- [x] 不做安装包
- [x] 不做自动清理

## Verification

- [x] focused tests passed
- [x] full tests passed
- [x] solution build passed
- [x] independent review passed
